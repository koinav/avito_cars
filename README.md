# Парсинг данных о машинах с Авито, обогащение и анализ данных

## Описание проекта

Данный проект демонстрирует полный цикл работы с данными в области автомобильного рынка. В рамках проекта реализованы следующие этапы:
- **Скраппинг данных**: сбор информации о машинах с сайта Авито.
- **Обогащение данных**: интеграция с внешними API для получения дополнительной информации о транспортных средствах.
- **Эксплоративный анализ (EDA)**: визуализация и статистическая обработка данных для выявления закономерностей и трендов.

Проект выполнен в образовательных целях и направлен на развитие навыков работы с веб-данными, API и инструментами анализа данных.

## Содержание проекта

- **car_parsing.ipynb**  
  В этом ноутбуке реализован процесс веб-скраппинга данных с Авито. Код отвечает за формирование HTTP-запросов, парсинг HTML-страниц и сохранение полученной информации для дальнейшей обработки.

- **APi_and_preprocessing.ipynb**  
  Здесь производится обогащение данных с помощью API, что позволяет дополнить первоначально собранную информацию. Также реализована предварительная обработка данных: очистка, нормализация и преобразование, необходимые для корректного анализа.

- **EDA.ipynb**  
  Нотбук, посвящённый эксплоративному анализу данных. С его помощью выполняется построение графиков, расчет статистических сводок и выявление основных трендов и зависимостей в данных.

## Используемые технологии и библиотеки

- **Язык программирования**: Python  
- **Веб-скраппинг**: BeautifulSoup, Requests  
- **Обработка данных**: Pandas  
- **Визуализация**: Matplotlib, Seaborn (возможно, Plotly)  
- **API-интеграция**: работа с внешними API для обогащения данных  
- **Интерактивная разработка**: Jupyter Notebook
